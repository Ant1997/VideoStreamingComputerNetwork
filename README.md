# VideoStreamingComputerNetwork
Video Streaming Project 

4-Buttons Interface:
  Type cmd in the directory of all the python files. 
  Put in (Server): 
     py -3 Server.py 2100 
  Type cmd in the directory of all the python files.
  Put in: 
     py -3 ClientLauncher.py localhost 2100 12000 movie.Mjpeg 4
